# Institute of Digital Risk (IDR) вЂ“ Homepage Assignment

This project is a responsive homepage designed for the **Institute of Digital Risk (IDR)**.
The assignment focuses on building a clean and professional website using **HTML, CSS, and JavaScript** while following the provided design and technical requirements.

---

## Project Overview

The **Institute of Digital Risk (IDR)** is an industry-led training and deployment institute focused on digital, cyber, and AI risk.
This homepage introduces the organization, explains its mission, and highlights the services and community it supports.

The website is designed with a **minimalist tech and education-focused style** using the color palette **orange, black, and white**.

---

## Features

* Responsive design for **desktop and mobile**
* **Sticky navigation bar** with smooth scrolling
* Modern **hero section** with call-to-action
* Informative **About section**
* **Service pillars** displayed using card layout
* **Community section** describing target audience
* **Contact / Register Interest form**
* Clean UI with hover effects and accessible color contrast

---

## Logo Design

The IDR logo uses a **cube-inspired geometric icon** to represent structure, resilience, and digital risk frameworks.

**Design choices:**

* **Cube Shape:** Symbolizes layered digital infrastructure and structured risk management.
* **Orange Color:** Represents innovation, energy, and forward-thinking technology.
* **Black & White:** Provides professionalism, clarity, and strong contrast.
* **Minimalist Style:** Ensures the logo remains legible across different sizes, including mobile headers and favicons.

Two variants were created:

* Icon-only logo
* Icon with text: *Institute of Digital Risk*
